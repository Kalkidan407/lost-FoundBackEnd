## Authentication Flow 

  1. User register via  
     ```
      POST /api/auth/register
     ```

 2. User login via 
  ```
    POST /api/auth/login
  ``` 
 3. Server returns JWT token 
 ```
   {
     "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
```


4. Use token in requests
   
   ```
   Authorization: Bearer <your_token>
   ```
